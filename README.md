# coonutils

## This is a C#-Libarary based on .Net 4.6.1 containing classes for:
* AES En-/Decryption
* CIFS-(Samba)-Client
* FTP-Client
* Windows Eventlogging
* TCP-Socket-Client
* Windows Registry Access
* FileSearcher

## Help:
### How to use:
Simply open Project in Visual Studio and compile.
Then use created .dll-File in your Project.
